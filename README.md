#### Installation

```
npm install --global zcui
zcui init HelloWorld --target vue
```

```
cd HelloWorld
```

#### Available commands

```
# start development server
$ npm run dev

# create production build
$ npm run prod

#analyze dependencies
$ npm run analyze
```
